# EasyBuy
一个javaweb项目，实现用户登录和注册功能，商品信息页面展示和购物车结算，商品信息的三级嵌套展示和管理员登录查看管理用户信息等功能
Servlet做后端业务层服务层的请求处理，JDBC连接MySQL数据库，ajax异步请求做页面不需重新加载就更新页面信息等技术支持
